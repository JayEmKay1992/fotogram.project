# Fotogram 📸
Ein barrierefreies, responsives Fotoalbum-Projekt.

## Features
- **Fluid Design:** Nutzt CSS `clamp()` und CSS Grid für stufenlose Skalierung.
- **Accessibility (WCAG):** 100% Lighthouse-Accessibility-Score, inklusive Focus-Trapping, Tastaturbedienung und ARIA-Labels.
- **Interaktivität:** Dynamisches Laden von Bildern mit JavaScript, eine voll funktionsfähige Lightbox (Modal) und ein interaktives Like-System mit persistenter Speicherung (localStorage).
